# VS Code üçün Azərbaycan Dil Paketi

Azərbaycan Dil Paketi, VS Code üçün lokallaşdırılmış istifadəçi interfeysi təcrübəsi təqdim edir.

## İstifadə

"İnterfeys Dilini Konfiqurasiya Et" əmri ilə VS Code interfeys dilini açıq şəkildə təyin edərək, standart istifadəçi interfeysi dilini dəyişə bilərsiniz.  
"Komanda Palitrası" alətini açmaq üçün "Ctrl+Shift+P" düyməsinə basın və "İnterfeys Dilini Konfiqurasiya Et" əmri üçün filtr tətbiq etmək və göstərmək üçün "dil" yazmağa başlayın. "Enter" düyməsinə basdığınızda, vurğulanan yerli ayarlar ilə birlikdə yerli ayarlara görə yüklənmiş dillərin siyahısı göstəriləcək. İstifadəçi interfeysi dilini dəyişmək üçün başqa bir "yerli ayar" seçin.  
Daha ətraflı məlumat üçün [Sənədlər](https://go.microsoft.com/fwlink/?LinkId=761051)-ə baxın.

## Tövhə Verən

Tərcümə təkmilləşdirməsi ilə bağlı rəy bildirmək üçün zəhmət olmasa [vscode-loc](https://github.com/microsoft/vscode-loc) deposunda Mövzu yaradın.  
Tərcümə sətirləri Microsoft Lokallaşdırma Platformasında saxlanılır. Dəyişiklik yalnız Microsoft Lokallaşdırma Platformasında edilə bilər, daha sonra vscode-loc deposuna ötürülür. Bu səbəbdən, vscode-loc deposunda "pull request" qəbul edilmir.

# VS Code Azerbaijani Language Pack

The Azerbaijani Language Pack provides a localized user interface experience for VS Code.

## Usage

You can override the default UI language by explicitly setting the VS Code display language using the **Configure Display Language** command.  
Press `Ctrl+Shift+P` to open the **Command Palette**, then start typing `language` to filter and display the **Configure Display Language** command.  
When you press `Enter`, a list of installed languages by locale will appear, with the current locale highlighted. Select another `locale` to switch the UI language.  
For more information, see [Documentation](https://go.microsoft.com/fwlink/?LinkId=761051).
